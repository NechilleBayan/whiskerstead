HOW TO USE THE MARKERS
  [ ]      = needed, not yet generated
  [GEN]    = generated, not yet imported into assets/
  [ADDED]  = PNG is present in the project (both 2x/ and 1x/ folders)
  Flip a row to [ADDED] when the files land — that is your "it's in the
  project" mark. Batch 1 rows are also listed in BATCH-1-PROMPTS.txt.

ENGINE COLUMN
  READY = the game loads this from disk today (or immediately after the
          already-approved wiggle step 1 lands). Drop the file in, it shows up.
  CODE  = renderer change required before this art can be consumed (today it
          is drawn procedurally in src/render/canvas-renderer.ts). Generate
          whenever you like, but it will not appear in-game until wired.

Every name below is authored at 2x (256x256 cats, see SPRITE-SPEC-
REFERENCE.txt for other categories) and exported to BOTH assets/<cat>/2x/
and /1x/ under the same filename. One row = one filename = two exported files.
